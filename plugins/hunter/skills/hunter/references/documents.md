# Documents and Career Materials

Create, tailor, validate, deliver, upload, and maintain career materials using
the strongest artifact capabilities available in the host.

## Covered materials

- resumes, CVs, professional profiles, biographies, and portfolio copy;
- cover letters, application answers, and supporting statements;
- recruiter introductions, outreach, and follow-ups;
- contract proposals, capability statements, and consulting materials;
- interview story banks, preparation briefs, and negotiation messages; and
- any other profile, portal, or opportunity-specific career content.

## Build from the selected profile

Resolve the active profile and relevant opportunity, pursuit, or relationship.
Use the user's source material and connected context. Cross-profile reuse is
allowed only when the user requests it.

Create directly from scratch when needed; an existing resume is not required.
Keep names, dates, metrics, links, and claims aligned with the working profile
and the user's instructions.

## Use available artifact tools

Prefer an editable/downloadable artifact when the host can create one. Inspect
structure and, when available, the rendered result for readability, clipping,
spacing, page flow, and visual hierarchy. Otherwise provide complete copy-ready
content rather than an outline.

When the user asks to upload a resume, update a professional profile, complete
application answers, or place content into a portal, use the corresponding host
capability. Follow the host's confirmation and authentication flow.

Preserve imported originals unless the user requests replacement. Create a
targeted version by default.

## Record actual status

Distinguish content prepared, file created, file opened, render inspected,
uploaded, profile updated, and message sent. Use the host's returned result for
each completed claim.

When useful, store artifact metadata under its owning profile with links to the
relevant opportunity, pursuit, and relationships. If portable storage is not
available, return the file or complete content directly.
